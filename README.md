# 🎵 NaviSpot (v1.0-alpha)

**NaviSpot** es un cliente de escritorio moderno para servidores de música **Navidrome**. Construido con Python y CustomTkinter, ofrece una experiencia fluida para disfrutar de tu biblioteca personal con un diseño premium.

> ⚠️ **Estado del Proyecto:** Esta es la **v1** y todavía se encuentra en desarrollo activo. Algunas funciones pueden estar incompletas o presentar errores.

---

## ✨ Características actuales
* **Interfaz Premium:** Diseño oscuro basado en la experiencia de usuario de Spotify.
* **Conexión Navidrome:** Integración nativa con la API de Navidrome/Subsonic.
* **Exploración:** Visualización de álbumes y artistas con portadas en alta resolución.
* **Reproductor Completo:** Controles de reproducción, barra de progreso, volumen logarítmico y modo aleatorio/repetir.

## 🚀 Próximamente (Roadmap)
- 🔍 Barra de búsqueda funcional.
- 📜 Gestión de Playlists.
- 📄 Visualización de letras de canciones.
- ⬇️ Descarga de pistas para reproducción offline.
- 🌓 Soporte para temas personalizados.

## 🛠️ Instalación

1. **Clonar el repositorio:**
   ```
   git clone https://github.com/nestoree/NaviSpot
   cd NaviSpot
   ```

2. **Instalar dependencias:**
Asegúrate de tener Python 3.10+ instalado.
```
pip install customtkinter Pillow requests pygame
```

3. **Ejecutar la aplicación:**
```
python main.py
```

## ⚙️ Configuración
Al iniciar la aplicación por primera vez, se te pedirá que introduzcas:

- URL del servidor: (ej: http://tu-servidor:4533)
- Usuario: Tu nombre de usuario de Navidrome.
- Contraseña: Tu contraseña (se genera un token seguro para la conexión).
